# Passato prossimo

Il **passato prossimo** è il tempo più usato per raccontare azioni concluse
nel passato (*Perfekt*). Corrisponde spesso al tedesco *ich habe gegessen* /
*ich bin gegangen*.

## Come si forma

**Presente di *avere* o *essere*** + **participio passato**.

Esempi: io **ho** mangi**ato**, io **sono** and**ato**.

## Il participio passato regolare

| Infinito | Terminazione | Participio |
|---|---|---|
| -are | **-ato** | mangi**are** → mangi**ato** |
| -ere | **-uto** | cad**ere** → cad**uto** |
| -ire | **-ito** | cap**ire** → cap**ito** |

## Participi passati irregolari comuni

| Infinito | Participio |
|---|---|
| fare | **fatto** |
| dire | **detto** |
| leggere | **letto** |
| scrivere | **scritto** |
| prendere | **preso** |
| vedere | **visto** |
| bere | **bevuto** |
| aprire | **aperto** |
| essere | **stato** |
| venire | **venuto** |
| mettere | **messo** |
| chiudere | **chiuso** |
| perdere | **perso** |
| scegliere | **scelto** |
| vincere | **vinto** |

## *Essere* o *avere*?

La maggior parte dei verbi usa **avere**. Si usa **essere** con:

- **verbi di movimento e stato**: andare, venire, entrare, uscire, salire,
  scendere, partire, arrivare, stare, rimanere, restare;
- **verbi di cambiamento**: nascere, morire, diventare;
- **verbi riflessivi/pronominali**: svegliarsi, alzarsi, lavarsi…

!!! warning "L'accordo con *essere*"
    Con **essere** il participio si accorda in **genere e numero** con il soggetto:

    - Marco è and**ato** / Maria è and**ata**
    - I ragazzi sono and**ati** / Le ragazze sono and**ate**

    Con **avere** il participio di solito **non** cambia:
    *ho mangiato*, *abbiamo mangiato*.

## Esempi

- Io **ho mangiato** una pizza. (Ich habe eine Pizza gegessen.)
- Ieri **ho letto** un libro interessante.
- Io **sono andato** al bar con gli amici.
- Maria **è partita** per Roma.
- Stamattina mi **sono svegliato** presto. (riflessivo → essere)
- Noi **siamo rimasti** a casa tutto il giorno.

## Esercizi

Metti le frasi al passato prossimo. Scegli *essere* o *avere* e accorda il participio.

1. Io (mangiare) ___ una mela.
2. Tu (scrivere) ___ una lettera.
3. Maria (andare) ___ al mercato.
4. Noi (vedere) ___ un bel film.
5. I ragazzi (partire) ___ ieri.
6. Voi (fare) ___ i compiti?
7. Lei (nascere) ___ a Milano.
8. Io mi (svegliare) ___ tardi.
9. Loro (prendere) ___ il treno.
10. Le ragazze (rimanere) ___ a casa.

??? success "Soluzioni"
    1. ho mangiato
    2. hai scritto
    3. è andata
    4. abbiamo visto
    5. sono partiti
    6. avete fatto
    7. è nata
    8. mi sono svegliato/a
    9. hanno preso
    10. sono rimaste
