# Aggettivi qualificativi

Gli aggettivi qualificativi descrivono un nome (una cosa, una persona). In italiano l'aggettivo **si accorda** (stimmt überein) con il nome in **genere** (maschile/femminile) e **numero** (singolare/plurale).

## Accordo dell'aggettivo

### Aggettivi in -o (4 forme)

Questi aggettivi hanno quattro forme: **-o / -a / -i / -e**.

| | Singolare | Plurale |
|-----------|-----------|---------|
| Maschile  | alt**o**  | alt**i** |
| Femminile | alt**a**  | alt**e** |

Esempi: un ragazzo alt**o**, una ragazza alt**a**, i ragazzi alt**i**, le ragazze alt**e**.

### Aggettivi in -e (2 forme)

Questi aggettivi hanno solo due forme: **-e** (singolare) / **-i** (plurale). Non cambiano tra maschile e femminile.

| | Singolare | Plurale |
|-----------|-----------|---------|
| Maschile  | grand**e** | grand**i** |
| Femminile | grand**e** | grand**i** |

Esempi: un libro grand**e**, una casa grand**e**, i libri grand**i**, le case grand**i**.

!!! note "Nota"
    L'aggettivo in italiano di solito segue il nome: *una macchina rossa* (nicht *una rossa macchina*).

## Superlativo assoluto

Il superlativo assoluto indica una qualità al massimo grado, senza confronto. Significa **"molto + aggettivo"**.

Si forma togliendo l'ultima vocale e aggiungendo **-issimo / -issima / -issimi / -issime**.

- bella → **bellissima** (= molto bella)
- alto → **altissimo** (= molto alto)
- grande → **grandissimo** (= molto grande)
- veloci → **velocissimi** (= molto veloci)

!!! tip "Trucco"
    Il superlativo assoluto si accorda come un normale aggettivo in -o: bell**issimo** / bell**issima** / bell**issimi** / bell**issime**.
