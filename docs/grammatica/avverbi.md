# Avverbi di modo

Gli avverbi di modo dicono **come** si fa un'azione (wie?). In italiano molti si formano aggiungendo **-mente** all'aggettivo.

## Formazione

### Regola base: aggettivo femminile + -mente

Si parte dalla forma **femminile** dell'aggettivo e si aggiunge **-mente**.

- lento → lent**a** → **lentamente**
- sicuro → sicur**a** → **sicuramente**

### Aggettivi in -e

Gli aggettivi che finiscono in **-e** (uguali al maschile e femminile) aggiungono direttamente **-mente**.

- veloce → **velocemente**

### Aggettivi in -le / -re

Gli aggettivi che finiscono in **-le** o **-re** (dopo una vocale) **perdono la -e finale** prima di **-mente**.

- facile → facil~~e~~ → **facilmente**
- gentile → gentil~~e~~ → **gentilmente**
- regolare → regolar~~e~~ → **regolarmente**

!!! warning "Attenzione"
    Solo -le / -re preceduti da vocale perdono la -e: *facile → facilmente*, ma *folle → follemente* (dopo consonante si mantiene).

## Avverbi irregolari

Due avverbi molto comuni **non** si formano con -mente:

| Aggettivo | Avverbio |
|-----------|----------|
| buono     | **bene** (gut) |
| cattivo   | **male** (schlecht) |

## Aggettivo o avverbio?

L'**aggettivo** descrive un nome. L'**avverbio** descrive un verbo.

- Anna è una **brava** cantante. *(aggettivo → descrive "cantante")*
- Anna canta **bene**. *(avverbio → descrive "canta")*

!!! tip "Trucco"
    Se puoi chiedere *"com'è?"* → aggettivo. Se puoi chiedere *"come fa?"* → avverbio.
