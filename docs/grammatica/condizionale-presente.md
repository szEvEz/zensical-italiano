# Condizionale presente

Il **condizionale presente** si usa per esprimere desideri, richieste gentili,
consigli e situazioni ipotetiche. In tedesco corrisponde spesso a *würde / hätte gern*,
in inglese a *would*.

## Quando si usa

- **Desideri** → *Vorrei* un caffè. (Ich hätte gern einen Kaffee.)
- **Richieste cortesi** → *Potresti* aiutarmi, per favore?
- **Consigli** → *Dovresti* riposare di più.
- **Ipotesi** (periodo ipotetico) → Se avessi tempo, *viaggerei* di più.
  → vedi [congiuntivo imperfetto](congiuntivo-imperfetto.md)
- **Notizia non confermata** → Il treno *arriverebbe* alle nove.

## Come si forma

Si parte dalla **radice del futuro** e si aggiungono le desinenze:

| Persona | Desinenza |
|---|---|
| io | **-ei** |
| tu | **-esti** |
| lui/lei | **-ebbe** |
| noi | **-emmo** |
| voi | **-este** |
| loro | **-ebbero** |

!!! note "Attenzione ai verbi in **-are**"
    Nei verbi in **-are** la *a* diventa *e*: parl**are** → parl**er**ei
    (come nel futuro).

## Verbi regolari

| | parl**are** | prend**ere** | dorm**ire** |
|---|---|---|---|
| io | parlerei | prenderei | dormirei |
| tu | parleresti | prenderesti | dormiresti |
| lui/lei | parlerebbe | prenderebbe | dormirebbe |
| noi | parleremmo | prenderemmo | dormiremmo |
| voi | parlereste | prendereste | dormireste |
| loro | parlerebbero | prenderebbero | dormirebbero |

## Verbi irregolari più comuni

La radice è la stessa del **futuro semplice**. Ecco la forma **io** (le desinenze
sono sempre -ei, -esti, -ebbe, -emmo, -este, -ebbero):

| Verbo | io (condizionale) |
|---|---|
| essere | **sarei** |
| avere | **avrei** |
| fare | **farei** |
| dare | **darei** |
| stare | **starei** |
| andare | **andrei** |
| dovere | **dovrei** |
| potere | **potrei** |
| sapere | **saprei** |
| vedere | **vedrei** |
| vivere | **vivrei** |
| volere | **vorrei** |
| venire | **verrei** |
| rimanere | **rimarrei** |
| bere | **berrei** |

## Esempi

- **Vorrei** affittare un appartamento a Milano dal 15 al 25 agosto.
- **Vorrei** comprare un biglietto per il Vietnam.
- **Vorrei** prenotare un tavolo per due persone alle 20.
- Mi **interesserebbe** saperne un po' di più.
- Che cosa ti **piacerebbe** sapere?
- Io **dovrei** trovare una soluzione, ma non l'ho ancora trovata.

## Esercizi

Coniuga il verbo tra parentesi al condizionale presente.

1. (io – volere) __________ un gelato.
2. (tu – potere) __________ chiudere la finestra?
3. (lui – dovere) __________ studiare di più.
4. (noi – andare) __________ volentieri al mare.
5. (voi – essere) __________ così gentili da aspettare?
6. (loro – mangiare) __________ volentieri una pizza.
7. (io – avere) __________ bisogno di aiuto.
8. (tu – venire) __________ con me domani?
