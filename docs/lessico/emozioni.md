# Le emozioni

Per parlare di come stai, usa l'espressione **Mi sento…** (Ich fühle mich…) seguita da un aggettivo.

| Mi sento… | Tedesco |
|------------|-----------|
| felice | glücklich |
| triste | traurig |
| arrabbiato/a | wütend |
| tranquillo/a | ruhig |
| preoccupato/a | besorgt |
| entusiasta | begeistert |
| stanco/a | müde |
| rilassato/a | entspannt |
| nervoso/a | nervös |
| orgoglioso/a | stolz |
| deluso/a | enttäuscht |
| curioso/a | neugierig |
| spaventato/a | verängstigt |
| soddisfatto/a | zufrieden |
| confuso/a | verwirrt |
| emozionato/a | aufgeregt |
| annoiato/a | gelangweilt |
| sollevato/a | erleichtert |
| grato/a | dankbar |
| fiducioso/a | zuversichtlich |

!!! note "Nota — l'accordo -o / -a"
    Molti aggettivi cambiano la vocale finale in base al genere di chi parla:

    - Un uomo dice: *Mi sento stanc**o**, arrabbiat**o**, grat**o**.*
    - Una donna dice: *Mi sento stanc**a**, arrabbiat**a**, grat**a**.*

    Gli aggettivi in **-e** come **felice**, **triste** ed **entusiasta** restano uguali per maschile e femminile.

!!! tip "Trucco"
    Puoi aggiungere un motivo con **perché**: *Mi sento felice **perché** oggi è sabato.*

## Esercizi

1. Come ti senti oggi? Rispondi con "Mi sento…".
2. Completa (per una donna): "Mi sento stanc__ e nervos__."
3. Qual è il contrario di *triste*?

??? success "Soluzioni"
    1. Risposta libera, ad esempio: "Mi sento rilassato/a perché è domenica."
    2. Mi sento stanc**a** e nervos**a**.
    3. **felice** (o *contento/a*).
