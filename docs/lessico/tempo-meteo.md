# Il tempo (meteo)

Per chiedere che tempo fa si dice:

- **Com'è il tempo?** (Wie ist das Wetter?)
- **Che tempo fa?**

## Vocabolario

| Italiano | Tedesco / spiegazione |
|-----------|-------------------------|
| Il tempo è bello. | Das Wetter ist schön. |
| Il tempo è brutto. | Das Wetter ist schlecht. |
| Il tempo è buono. | Das Wetter ist gut. |
| Il tempo è cattivo. | Das Wetter ist schlecht (wörtlich: „böse"). |
| C'è il sole. (è soleggiato) | Die Sonne scheint. (sonnig) |
| Ci sono le nuvole. (è nuvoloso) | Es ist bewölkt. |
| C'è la pioggia. (piove) | Es regnet. |
| C'è la neve. (nevica) | Es schneit. |
| Fa caldo. | Es ist warm/heiß. |
| Fa freddo. | Es ist kalt. |

!!! note "Nota"
    - **C'è** + singolare (c'è il sole, c'è la neve), **ci sono** + plurale (ci sono le nuvole).
    - **Piove**, **nevica** sono verbi impersonali: si usano solo alla terza persona singolare.
    - Per la temperatura si usa **fa**: *fa caldo*, *fa freddo*, *fa fresco*.

!!! tip "Trucco"
    Puoi collegare il tempo alla stagione: *In inverno fa freddo e a volte nevica. In estate fa caldo e c'è il sole.*

## Esercizi

1. Traduci: "Es regnet und es ist kalt."
2. Completa: "___ le nuvole, è nuvoloso."
3. Rispondi: "Com'è il tempo oggi?"

??? success "Soluzioni"
    1. Piove e fa freddo. / C'è la pioggia e fa freddo.
    2. **Ci sono** le nuvole, è nuvoloso.
    3. Risposta libera, ad esempio: "Oggi il tempo è bello, c'è il sole e fa caldo."
