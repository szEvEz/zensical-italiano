# Gli animali

Ogni animale è descritto con **un aggettivo**. Fai attenzione all'accordo (maschile/femminile) tra il nome e l'aggettivo!

| Animale | Descrizione |
|----------|--------------|
| il cavallo | è veloce (schnell) |
| il pesce | è piccolo (klein) |
| il gatto | è indipendente (unabhängig) |
| la volpe | è intelligente (schlau) |
| la tartaruga | è lenta (langsam) |
| il lupo | è morbido (weich) |
| il delfino | è socievole (gesellig) |
| il cane | è amichevole (freundlich) |
| lo squalo | è aggressivo (aggressiv) |
| il leone | è il re (der König) |
| la pecora | è calma (ruhig) |
| la scimmia | è divertente (lustig) |
| l'elefante | è grande (groß) |
| il serpente | è velenoso (giftig) |
| la capra | è erbivora (pflanzenfressend) |
| l'orso | è goloso (naschhaft) |
| il coccodrillo | è aggressivo (aggressiv) |
| il coniglio | è dolce (süß, sanft) |
| la mucca | è gentile (sanftmütig) |
| il toro | è arrabbiato (wütend) |
| la zebra | è a strisce (gestreift) |
| l'uccello | è rumoroso (laut) |
| il ragno | è disgustoso (ekelhaft) |
| la giraffa | è alta (hoch, groß) |

!!! note "Nota — l'accordo dell'aggettivo"
    L'aggettivo cambia in base al genere del nome:

    - la pecora è calm**a** / il gatto è calm**o**
    - la volpe è intelligent**e** / il cane è intelligent**e** (gli aggettivi in **-e** non cambiano al singolare)

!!! tip "Trucco"
    Nota gli articoli particolari: **lo** squalo (davanti a *s* + consonante), **l'**elefante, **l'**orso, **l'**uccello (davanti a vocale).

## Esercizi

1. Completa con l'aggettivo giusto: "La tartaruga è ___ (lento)."
2. Qual è "il re" degli animali?
3. Scegli un animale e descrivilo con un aggettivo diverso da quello della tabella.

??? success "Soluzioni"
    1. La tartaruga è **lenta**.
    2. Il leone.
    3. Risposta libera, ad esempio: "Il gatto è **pigro**."
