# Dire l'ora

Per chiedere l'ora puoi dire:

- **Che ore sono?** (Wie viel Uhr ist es?)
- **Che ora è?** (stessa domanda, al singolare)

## Esempi

| Ora | Come si dice |
|------|----------------|
| 9:43 | Sono le nove e quarantatré. |
| 10:30 | Sono le dieci e mezza (o: e trenta). |
| 5:00 | Sono le cinque. |
| 19:00 | Sono le diciannove. / Sono le sette (di sera). |
| 12:00 | È mezzogiorno. |
| 24:00 / 00:00 | È mezzanotte. |
| 1:00 | È l'una. |

!!! note "Nota — «è» o «sono le»?"
    Si usa il **singolare** con:

    - **È** l'una.
    - **È** mezzogiorno.
    - **È** mezzanotte.

    In tutti gli altri casi si usa il **plurale**: **Sono le** due, **Sono le** dieci e mezza…

!!! tip "Trucco"
    - **e mezza / e mezzo** = :30
    - **e un quarto** = :15
    - **meno un quarto** = :45 (Es. *le otto meno un quarto* = 7:45)

## A che ora…?

Per chiedere l'orario di un evento usa **a che ora**:

- **A che ora** apre il negozio? — Apre **alle** nove.
- **A che ora** ti alzi? — Mi alzo **alle** sette.
- **A che ora** è la lezione? — **All'**una. / **A** mezzogiorno.

!!! note "Nota"
    Con l'orario usa **alle** (alle otto), ma **all'una**, **a** mezzogiorno, **a** mezzanotte.

## Esercizi

1. Scrivi in parole: 8:00, 1:00, 12:00.
2. Completa: "___ ora è? — ___ l'una."
3. Rispondi: "A che ora fai colazione?"

??? success "Soluzioni"
    1. Sono le otto; È l'una; È mezzogiorno (12:00 di giorno) / È mezzanotte (00:00).
    2. **Che** ora è? — **È** l'una.
    3. Risposta libera, ad esempio: "Faccio colazione **alle** sette e mezza."
