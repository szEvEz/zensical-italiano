# Gli aggettivi di personalità

Questi aggettivi descrivono il **carattere** di una persona. Per ognuno trovi il significato e un breve esempio.

| Aggettivo | Significato ed esempio |
|------------|--------------------------|
| generoso | dà volentieri agli altri — *Luca è generoso: offre sempre il caffè.* |
| altruista | pensa al bene degli altri — *Anna è altruista: aiuta chi è in difficoltà.* |
| divertente | fa ridere — *Marco è divertente: racconta barzellette.* |
| timido | si vergogna facilmente — *Sara è timida: parla poco con gli sconosciuti.* |
| silenzioso | parla poco — *Il nonno è silenzioso: ascolta più che parlare.* |
| introverso | preferisce stare da solo — *Paolo è introverso: ama leggere in casa.* |
| studioso | ama studiare — *Giulia è studiosa: legge ogni giorno.* |
| irascibile | si arrabbia facilmente — *Paolo è **irascibile**: si arrabbia facilmente.* |
| socievole | sta volentieri con gli altri — *Elena è socievole: ha molti amici.* |
| estroverso | aperto e comunicativo — *Davide è estroverso: parla con tutti.* |
| amichevole | gentile e cordiale — *Il vicino è amichevole: saluta sempre.* |
| pigro | non ama fare fatica — *Il gatto è pigro: dorme tutto il giorno.* |
| puntuale | arriva sempre in orario — *Marta è puntuale: non è mai in ritardo.* |
| preciso | fa le cose con attenzione — *L'ingegnere è preciso nei calcoli.* |
| creativo | ha molte idee nuove — *L'artista è creativo: inventa storie.* |
| gentile | educato e cortese — *La maestra è gentile con i bambini.* |
| simpatico | piacevole da conoscere — *Il tuo amico è simpatico: mi piace.* |
| sincero | dice sempre la verità — *Franco è sincero: non mente mai.* |
| coraggioso | non ha paura — *Il pompiere è coraggioso.* |
| scherzoso | ama scherzare — *Il papà è scherzoso: fa sempre battute.* |
| ordinato | tiene tutto in ordine — *Lucia è ordinata: la sua scrivania è pulita.* |
| egoista | pensa solo a sé — *Non essere egoista: condividi con gli altri.* |
| intelligente | capisce in fretta — *Il bambino è intelligente: impara subito.* |
| sveglio | vispo e reattivo — *Quel ragazzo è sveglio: risolve tutto in fretta.* |
| sportivo | ama lo sport — *Mia sorella è sportiva: corre ogni mattina.* |
| allegro | di buon umore — *Il nonno è allegro: sorride sempre.* |
| maleducato | senza buone maniere — *È maleducato: non saluta mai.* |
| paziente | sa aspettare con calma — *La maestra è paziente con gli alunni.* |
| insicuro | dubita di sé — *Sono insicuro prima di un esame.* |
| debole | senza forza — *Dopo l'influenza mi sento debole.* |
| loquace | parla molto — *La zia è loquace: chiacchiera per ore.* |
| sensibile | prova emozioni intense — *Elena è sensibile: si commuove facilmente.* |
| permaloso | si offende per poco — *Non fare battute: è permaloso.* |
| deciso | sa cosa vuole — *Il capo è deciso: prende scelte rapide.* |

!!! note "Nota"
    Ricorda l'accordo: *Luca è generos**o***, *Anna è generos**a***. Gli aggettivi in **-e** (gentile, intelligente) e alcuni come **altruista**, **egoista** restano uguali al singolare per maschile e femminile.

!!! tip "Trucco"
    Molte coppie sono opposti: **introverso ↔ estroverso**, **generoso ↔ egoista**, **timido ↔ socievole**, **paziente ↔ irascibile**.

## Esercizi

1. Trova il contrario: *egoista*, *timido*, *pigro*.
2. Completa: "Marta arriva sempre in orario, è molto ___."
3. Descrivi un amico con tre aggettivi.

??? success "Soluzioni"
    1. egoista ↔ **generoso/altruista**; timido ↔ **socievole/estroverso**; pigro ↔ **attivo/sportivo**.
    2. Marta arriva sempre in orario, è molto **puntuale**.
    3. Risposta libera, ad esempio: "Il mio amico è simpatico, generoso e divertente."
