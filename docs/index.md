# Impara l'italiano 🇮🇹

Benvenuto! Questa è la mia **wiki personale** per imparare l'italiano: grammatica,
lessico, letture, dialoghi ed esercizi con soluzioni.

## Come è organizzata

<div class="grid cards" markdown>

- **[Conversazione](conversazione/index.md)**

    Domande e risposte per parlare di sé: presentarsi, famiglia, gusti, routine,
    lavoro, hobby e amicizia.

- **[Grammatica](grammatica/index.md)**

    I tempi verbali (presente, passato prossimo, imperfetto, condizionale…),
    i pronomi, le preposizioni, gli aggettivi e gli avverbi.

- **[Lessico](lessico/index.md)**

    Parole utili per argomento: giorni, ora, meteo, animali, vestiti, emozioni,
    personalità e parole avanzate.

- **[Letture](letture/index.md)** e **[Dialoghi](dialoghi/index.md)**

    Testi di comprensione (Catania, la tecnologia…) e dialoghi reali
    (al ristorante, in aereo, prenotazioni…).

</div>

## Come modificare la wiki (live edit)

Avvia il server locale con ricarica automatica: appena salvi un file `.md`,
la pagina si aggiorna nel browser.

``` bash
uv run zensical serve -o
```

!!! tip "Suggerimento"
    Tieni `zensical serve` aperto in un terminale, modifica i file dentro `docs/`
    e guarda il sito aggiornarsi in tempo reale.
