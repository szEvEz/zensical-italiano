# La famiglia

Parlare dei propri familiari: fratelli, sorelle, cosa fanno e dove vivono.

| Domanda | Come rispondere |
| --- | --- |
| Hai fratelli o sorelle? | Ho … fratelli e … sorelle. / Sono figlio/a unico/a. |
| Come si chiama tuo fratello? | Mio fratello si chiama … |
| Dove vive tuo fratello? | Mio fratello vive a … |
| Quanti anni ha tuo fratello? | Ha … anni. |
| Tuo fratello studia o lavora? | Studia … / Lavora, fa … |
| Quali sono i tuoi hobby? | I miei hobby sono … |

!!! tip "Suggerimento"
    Con i nomi di famiglia al singolare l'articolo si omette: *mio fratello*, *mia sorella*. Al plurale invece si usa: *i miei fratelli*, *le mie sorelle*.

Per le forme *ha*, *vive*, *lavora* vedi il [presente indicativo](../grammatica/presente-indicativo.md).
