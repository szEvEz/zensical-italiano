# Presentarsi

Le prime domande che senti quando conosci qualcuno. Esercitati a rispondere in modo naturale.

| Domanda | Come rispondere |
| --- | --- |
| Come ti chiami? | Mi chiamo … |
| Dove vivi? | Vivo a …, in … |
| Che fai nella vita? (studi o lavori?) | Studio … / Lavoro … |
| Quanti anni hai? | Ho … anni. |
| Perché vuoi imparare l'italiano? | Perché … |

!!! tip "Suggerimento"
    Ogni risposta si può adattare a te: cambia il nome, la città, il lavoro e l'età con i tuoi dati. Per esempio: *"Mi chiamo …, vivo a …, studio/lavoro …, ho … anni."*

Per rivedere le forme dei verbi (mi chiamo, vivo, lavoro, voglio), vedi il [presente indicativo](../grammatica/presente-indicativo.md).
