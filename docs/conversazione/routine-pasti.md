# La routine e i pasti

Parlare della giornata tipica: cosa mangi, a che ora fai le cose.

| Domanda | Come rispondere |
| --- | --- |
| Che cosa mangi normalmente a colazione? | A colazione mangio … e bevo … |
| A che ora fai colazione? | Faccio colazione alle … |
| Che cosa mangi normalmente a pranzo? | A pranzo mangio … |
| A che ora pranzi? | Pranzo alle … |
| Che cosa mangi normalmente a cena? | A cena mangio … |
| A che ora ceni? | Ceno alle … |
| A che ora ti svegli la mattina? | Mi sveglio alle … |
| A che ora vai a dormire? | Vado a dormire alle … |

!!! tip "Suggerimento"
    Per dire l'ora si usa *alle* + numero (*alle sette*, *alle venti*), ma con l'una si dice *all'una*. Vedi [dire l'ora](../lessico/ora.md).

I verbi *svegliarsi* (mi sveglio) sono riflessivi. Per le altre forme vedi il [presente indicativo](../grammatica/presente-indicativo.md).
