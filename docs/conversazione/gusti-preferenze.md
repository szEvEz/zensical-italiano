# Gusti e preferenze

Come parlare delle cose che preferisci: colori, cibo, città.

| Domanda | Come rispondere |
| --- | --- |
| Qual è il tuo colore preferito? | Il mio colore preferito è … |
| Qual è il tuo piatto preferito? | Il mio piatto preferito è … |
| Qual è la tua città preferita? | La mia città preferita è … |

!!! tip "Suggerimento"
    Nota il modello: **"Qual è …? Il mio / La mia … preferito/a è …"**

    L'aggettivo *preferito* concorda con il nome: *il colore preferit**o***, *la città preferit**a***. E ricorda: *qual è* si scrive senza apostrofo.
